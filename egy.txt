git init
inicializálod a repot

git status
a repo státuszát adja meg

git add .
minden fájlt hozzáad a repohoz

git commit -m "komment"
commitolod a repo-t a main branch-re egy kommenttel

git config --global --edit
az adataim(név és email) szerkesztése vim-mel
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

git log
git log --oneline
a commitekések "history"-ját írja ki
